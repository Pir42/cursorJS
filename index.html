Le script pour créer des curseurs personnalisés

Comment ça marche ?

Il vous suffit d'instancier la classe Cursor en passant en paramètres un tableau des éléments qui composeront votre curseur.

let mycursor = new Cursor([element1, element2])

Vous devez également introduire les règles de style suivantes dans vos feuilles de styles :


#cursor div {
    position: fixed;
    pointer-events: none;
    z-index: 9999;
}

Si vous voulez enlever le cursor par défaut vous devez utiliser la propriété none de cursor, et le déclarer dans un élément global tel que body

Composez vos éléments

Les éléments qui composent le curseur sont des objets. Les propriétés de cet objet définissent le style et le comportement de votre curseur. Voici un exemple d'élément de curseur : 


const myelement = {
    type: "circle",
    rayon: 20,
    stroke: {
        width: 1,
        color: "black",
    },
    fill: "#ff00ff",
    animate: {
        speed: .6,
        speedCurve: "cubic-bezier(0,.01,.34,1.01)",
    },
    hoverProperties: {
        scale: 0,
        rotate: 0
    },
    holdProperties: {
        scale: .8
    }
}

Concernant les propriétés plus en détail :

  • type: (string) Définit si le curseur est un cercle, par défaut c'est un carré
  • stroke: (object) Définit la bordure du curseur, si non-précisé il n'y aura pas de bordure
    • width (float) Définit la taille de la bordure
    • color (string) Définit la couleur de la bordure
  • fill: (string) Définit la couleur du curseur,
  • animate: (object) Définit la latence du curseur
    • speed (float) Définit la vitesse de latence
    • speedCurve (string) Définit les courbes de vitesse de la latence
  • hoverProperties: (object) Définit les propriétes de l'élément lors d'un survol des éléments définis
    • scale (float) Définit la nouvelle taille du curseur
    • rotate (float) Définit la nouvelle inclinaison du curseur
  • holdProperties: (object) Définit les propriétes du curseur lorsque le boutton gauche de la souris est maintenu
    • scale (float) Définit la nouvelle taille du curseur

Rajoutez des éléments avec lesquels interagir

Pour que les propriétés de survol marchent, il faut définir les éléments avec lesquels le curseur va interagir. La méthode addInteractElements vous permet de faire cela. Vous devez lui passer en paramètre un tableau des éléments séléctionnés

Astuce : vous pouvez utiliser des attributs de données à l'aide de data-*

                    
let mycursor = new Cursor([myelement]);

let interactElements = document.querySelectorAll('[data-hover]');

mycursor.addInteractElements(interactElements)

Autres effets

La propriété holdProperties vous permet de définir un état du curseur lorsque l'utilisateur maintient le clic gauche de sa souris. Pour activer ce comportement vous devez appeler la fonction holdCursor

                        
let mycursor = new Cursor([myelement]);

mycursor.holdCursor()
